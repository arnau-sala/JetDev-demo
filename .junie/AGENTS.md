# JetDev template constraints

- This project is a **minimal Vite + React** app.
- You may **only** edit `src/App.jsx` and `src/App.css` unless fixing an obvious build break in existing files.
- **Do not** add new npm dependencies (no Tailwind, no UI kits, no extra packages).
- **Do not** run `npm install` for new libraries. Styling is **plain CSS** (variables, flex, grid) and may use **Google Fonts** via `@import` in `App.css`.
- Keep the app buildable with `npm run build` after your changes.
